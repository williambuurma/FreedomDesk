# FreedomDesk Idea Vault

> **Purpose:** A parking lot for future ideas that do **not** belong in Version 1.  
> **Not the roadmap.** [`ROADMAP.md`](ROADMAP.md) schedules phased delivery. This document only captures ideas so they are not forgotten.

Ideas live here so they stay out of Version 1 implementation. Recording something in the vault is how we protect focus — not how we expand scope.

**Adding an idea to this document is not approval to build it.**

**Every idea must first pass the Decision Filter in [`V1_FOUNDATION.md`](V1_FOUNDATION.md) before entering active development.**

---

## How to use this vault

1. Capture the idea in the section below.
2. Leave Version 1 work unchanged.
3. When an idea is ready for serious evaluation, run it through the Decision Filter — then move it to [`FEATURE_BACKLOG.md`](FEATURE_BACKLOG.md) or [`ROADMAP.md`](ROADMAP.md) as appropriate.
